- Open class   
  - device master -> notify to all device slave
  - <master_emit> -> <slave_on> 

- Close class   
  - device master -> notify to all device slave
  - <master_emit> -> <slave_on>

- Slave join class   
  - device slave -> notify to master
  - <slave_emit> -> <master_on>

- Slave leave class   
  - device slave -> notify to master
  - <slave_emit> -> <master_on>
